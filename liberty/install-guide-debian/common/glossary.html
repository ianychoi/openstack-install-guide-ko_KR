용어집¶

이 용어집은 OpenStack에 관련된 개념에 대한 용어를 정의하고 정의된 용어의 목록을 제공합니다.

OpenStack 용어집에 추가하기 원한다면, openstack/openstack-manuals repository 를 클론 한뒤 doc/glossary/glossary-terms.xml 소스파일을 OpenStack 컨트리뷰션 프로세스에 맞춰 업로드하세요.

6to4

IPv6로의 마이그레이션을 제공하고 IPv4 네트워크를 통해 IPv6 패킷을 전송할 수 있도록 허용하는 메커니즘을 말합니다.

절대 임계치

게스트 VM에대한 가능한 임계치를 나타냅니다. 이 설정에는 총 RAM 크기, 최대 vCPU 개수, 최대 디스크 크기를 포함합니다.

접근 제어 목록

개체에 부여하는 권한 목록. ACL은 개체에 접근 가능한 사용자 또는 시스템 프로세스를 규정합니다. 또한 지정된 개체에 대해 어떤 작업이 수행 가능한지를 정의합니다. 일반적인 ACL 내 각 항목에는 제목 및 작업을 지정합니다. 예를 들면, 어떤 파일에 대한 ACL 항목 `(Alice, delete)`은 Alice에게 파일을 삭제하는 권한을 부여합니다.

접근 키

Amazon EC2 접근 키에 대한 다른 용어. EC2 접근 키 참조

계정

오브젝트 스토리지 컨텍스트에서의 계정입니다. Active Directory, /etc/passwd, OpenLDAP, OpenStack Identity 등 인증 서비스에서의 사용자 계정과 혼동하지 마십시오.

계정 관리자

백엔드 SQLite 데이터베이스에 대해 쿼리를 실행하여 지정된 오브젝트 저장 계정의 복제본 또는 잘못되거나 손상된 개체 누락을 확인합니다.

계정 데이터베이스

오브젝트 스토리지 계정 및 관련 메타데이터를 포함하고 계정 서버가 접근하는 SQLite 데이터베이스입니다.

계정 수거자

계정 데이터베이스에 대해 검색 및 삭제를 수행하고, 계정 서버가 삭제했음을 표시하는 오브젝트 스토리지 worker입니다.

계정 서버

오브젝트 스토리지 내 컨테이너 목록을 나열하고 컨테이너 정보를 계정 데이터베이스에 저장합니다.

계정 서비스

조회, 생성, 수정 및 감사와 같은 계정 서비스를 제공하는 오브젝트 스토리지 구성요소 입니다. OpenStack 인증 서비스, OpenLDAP, 또는 비슷한 사용자 계정 서비스와 혼동하지 마십시오.

어카운팅

Compute 서비스가 이벤트 통지 및 시스템 사용량 데이터 기능을 통해 계정 정보를 제공합니다.

ACL

접근 제어 목록을 살펴봅니다.

액티브/액티브 구성

액티브/액티브 구성으로 된 고가용성 설정에서는 여러 시스템들이 부하를 서로 공유하고, 하나가 실패하는 경우 해당 부하는 나머지 시스템에 분배됩니다.

Active Directory

Microsoft의 인증 및 identity 서비스로, LDAP 기반입니다. OpenStack에서 지원합니다.

액티브/패시브 구성

액티브/패시브 구성으로 된 고가용성 설정에서 실패한 부분을 온라인으로 대체 가능한 부가적인 자원들을 가져오도록 시스템을 설정합니다.

주소 풀

프로젝트에 할당되었거나, 프로젝트에 있는 VM 인스턴스에 할당되어 사용할 수 있는 고정 IP이거나 유동 IP 주소 그룹을 나타냅니다.

관리자 API

인증된 관리자에 의해 접근이 가능하고, 일반적으로 최종 사용자 또는 공용 인터넷에서는 접근 불가능한 API call의 부분 집합. 별도 서비스 (keystone) 또는 또 다른 API (nova)의 부분집합으로 존재 가능.

관리자 서버

인증 서비스 콘텍스트에서, admin API에 액세스를 제공하는 작업 프로세스입니다.

Advanced Message Queuing Protocol (AMQP)

내부 서비스 통신을 위한 OpenStack 구성요소로 사용되는 개방형 표준 메시징 프로토콜로, RabbitMQ, Qpid 또는 ZeroMQ가 제공함

Advanced RISC Machine (ARM)

모바일과 임베디드 장치에서 사용하는 저전력 CPU입니다. OpenStack 지원합니다.

알림

Compute 서비스는 사용자 정의 통지 드라이버를 생성할 수 있는 기능을 포함하는 통지 시스템을 통해 알림을 보낼 수 있습니다. 알림은 horizon 대시보드에서 보내거나 표시할 수 있습니다.

할당

주소 pool로부터 floating IP 주소를 가져오는 프로세스로 게스트 VM 인스턴스에서 fixed IP에 연계 가능함.

Amazon Kernel Image (AKI)

VM 컨테이너 포맷 및 디스크 포맷을 의미. Image 서비스에서 지원함.

Amazon Machine Image (AMI)

VM 컨테이너 포맷 및 디스크 포맷을 의미. Image 서비스에서 지원함.

Amazon Ramdisk Image (ARI)

VM 컨테이너 포맷 및 디스크 포맷을 의미. Image 서비스에서 지원함.

Anvil

Python으로 구성되어있는 DevStack을 셸 스크립트 기반 프로젝트의 이름입니다.

Apache

아파치 소프트웨어 재단은 오픈소스 소프트웨어 프로젝트의 아파치 커뮤니티를 지원합니다. 해당 프로젝트들은 공공의 이익을 위한 소프트웨어 제품을 제공합니다.

Apache License 2.0

모든 OpenStack 코어 프로젝트는 Apache License 2.0 라이센스하에 제공됩니다.

Apache 웹 서버

현재 인터넷에서 사용되고 있는 가장 일반적인 웹 서버 소프트웨어

API

응용프로그램 프로그래밍 인터페이스.

API 엔드 포인트

클라이언트가 API에 접근하기 위해 통신을 수행하는 데몬, 워커 또는 서비스. API 엔드 포인트는 인증, 데이터 세일즈, 성능 측정, Compute VM 명령어, 인구 조사 데이터 등과 같은 임의의 서비스에 제공합니다.

API 확장

일부 OpenStack 코어 API를 확장하는 사용자 정의 모듈을 말합니다.

API 확장 플러그인

네트워킹 플러그인이나 네트워킹 API 확장에 대한 대체 용어입니다.

API 키

API 토큰의 대체 용어입니다.

API 서버

API 엔드 포인트를 제공하는 데몬이나 워커를 실행하는 노드입니다.

API 토큰

API 요청을 전송하고, OpenStack을 사용하는데 있어 클라이언트가 요청한 작업을 할 수 있는 권한이 있는지 확인합니다.

API 버전

OpenStack에서, 프로젝트에 대한 API 버전이 URL 일부분으로 되어 있습니다. 예를 들면 :file:`example.com/nova/v1/foobar`와 같습니다.

애플릿

웹페이지 내에 포함할 수 있는 자바 프로그램

Application Programming Interface (API)

서비스, 응용 프로그램, 또는 프로그램에서 사용되는 명세에 대한 집합. 각 call에 대해 필요로 하는 파라메터에 해당하는 서비스 요청과 예상되는 리턴 값을 포함.

응용 프로그램 카탈로그

응용 프로그램 카탈로그 서비스를 제공하여 사용자가 응용 프로그램 수명주기를 관리하는 동안 응용 프로그램 추상화 레벨에서 복합 환경을 작성하고 배포 가능하도록 제공하는 OpenStack 프로젝트입니다. 프로젝트 코드 이름은 murano입니다.

어플리케이션 서버

네트워크 상의 다른 소프트웨어 부분을 활용 가능하게 만들어 주는 소프트웨어의 한 부분

어플리케이션 서비스 프로바이더 (ASP)

기업 및 기관에게 부가적인 서비스들을 보다 낮은 가격에 제공하도록 돕는 특화된 어플리케이션을 임대하는 회사

주소 변환 프로토콜 (ARP)

layer-3 IP 주소에서 layer-2 링크 로컬 주소로 변환되는 방법에 대한 프토토콜

arptables

Linux 커널 방화벽 모듈 내 주소 변환 프로토콜 패킷 필터 규칙을 유지하기 이해 사용하는 도구. Compute 내에서 방화벽 서비스를 VM에 제공하기 위해 iptables, ebtables 및 ip6tables와 함께 사용됨

연결

Compute floating IP 주소를 fixed IP 주소와 연결시키는 과정

비동기 JavaScript와 XML (AJAX)

비동기 웹 어플리케이션을 제작하기 위해 클라이언트 사이드에서 사용되는 상호 관련이 있는 웹 개발 테크닉의 모음. horizon에서 광범위하게 사용됨.

ATA over Ethernet (AoE)

Ethernet 내 터널링되어 있는 디스크 스토리지 프로토콜

연결

네트워킹에서 L2 네트워크와 vNIC이나 VIF의 연결 프로세스를 나타냅니다. Compute의 콘텍스트에서 이 프로세서는 인스턴스 스토리지 볼륨과 연결됩니다.

attachement (네트워크)

논리적 포트에 인터페이스 ID 모음. 포트에 인터페이스를 연결합니다.

auditing

시스템이 사용하는 데이터 기능을 사용해서 Compute에 제공합니다.

auditor

오프젝트 스토리지 오브젝트, 컨테이너, 계정의 무결성을 검사하는 프로세스입니다. Auditors 는 오브젝트 스토리지 계정 auditor, 컨테이너 auditor, 오브젝트 auditor에 대한 전체를 다 말합니다.

Austin

OpenStack의 초기 릴리즈 코드 이름. 첫 디자인 서밋을 한 곳인 미국 텍사스 Austin을 나타냅니다.

auth 노드

오브젝트 스토리지 authorization 노드에대한 대체 용어입니다.

인증

사용자, 프로세스 또는 클라인트 중 정말 누구가 말하는지를 개인 키, 암호화된 토큰, 암호, 지문과 비슷한 매소드들을 통해 확인하는 프로세스를 말합니다.

인증 토큰

인증후 클라이언트에 제공하는 텍스트 문자열입니다. API 엔드 포인트에서 이후에 요청된 프로세스나 사용자에의해 제공되어야합니다.

AuthN

인증 서비스를 제공하는 Identity 서비스 컴포넌트

허가

사용자, 프로세스, 클라이언트가 어떤 행동을 수행하는 데 있어 권한이 부여되었는지를 검증하는 것

허가 노드

허가 서비스를 제공하는 오브젝트 스토리지 노드

AuthZ

높은 수준의 인증 서비스를 제공하는 Identity 구성요소.

자동 ACK

메시지를 수락할 것인지에 대해 활성화 또는 비활성화를 RabbitMQ 내에서 구성하며, 디폴트로 활성화됨

자동 선언

메시지 교환 여부를 결정하는 Compute RabbitMQ 설정은 프로그램이 시작할 때 자동으로 생성됩니다.

가용 구역

Amazon EC2 컨셉에 따른 결함 포용을 위한 격리된 지역. OpenStack Compute 존 또는 셀과 혼동하지 마십시오.

AWS

Amazon 웹 서비스.

AWS CloudFormation 템플릿

AWS CloudFormation은 AWS 사용자들이 관련된 리소스 집합을 생성 및 관리를 가능하게 합니다. Orchestration 서비스는 CloudFormation과 호환되는 포맷 (CFN)을 지원합니다.

백엔드

Compute 볼륨 마운트, 데몬에 의한 iSCSI 대상으로의 데이터 전송, 또는 Object Storage 개체 무결성 확인과 같은 사용자에게 있어 파악하기 어려운 상호작용 및 프로세스

백엔드 카탈로그

클라이언트가 사용 가능한 API 엔드포인트에 대한 정보를 검색하고 저장하는 Identity 서비스 카탈로그 서비스에서 사용되는 스토리지 방식입니다. 예시로는 SQL 데이터베이스, LDAP 데이터베이스, 또는 KVS 백엔드가 있습니다.

백엔드 저장소
The persistent data store used to save and retrieve information for a service, such as lists of Object Storage objects, current state of guest VMs, lists of user names, and so on. Also, the method that the Image service uses to get and store VM images. Options include Object Storage, local file system, S3, and HTTP.
backup restore and disaster recovery as a service

백업, 복구, 그리고 파일시스템, 인스턴스 또는 데이터베이스 백업에 대한 복원을 위한 통합 툴을 제공하는 OpenStack 프로젝트입니다. 프로젝트 이름은 freezer입니다.

대역폭

인터넷과같은 통신 자원에 의해 사용되는 가용한 데이터의 양 입니다. 다운로드에 사용되는 데이터의 양 또는 다운로드 가능한 데이터의 양을 나타냅니다.

barbican

OpenStack을 위한 키 관리 서비스의 코드 이름입니다.

bare
An Image service container format that indicates that no container exists for the VM image.
베어메탈 서비스

가상머신과 상반되는, 베어메탈을 프로비저닝하는 OpenStack 프로젝트. 프로젝트 코드 이름은 ironic입니다.

기본 이미지

OpenStack에서 제공하는 이미지.

Bell-LaPadula 모델
A security model that focuses on data confidentiality and controlled access to classified information. This model divide the entities into subjects and objects. The clearance of a subject is compared to the classification of the object to determine if the subject is authorized for the specific access mode. The clearance or classification scheme is expressed in terms of a lattice.
Benchmark service
OpenStack project that provides a framework for performance analysis and benchmarking of individual OpenStack components as well as full production OpenStack cloud deployments. The code name of the project is rally.
Bexar
A grouped release of projects related to OpenStack that came out in February of 2011. It included only Compute (nova) and Object Storage (swift). Bexar is the code name for the second release of OpenStack. The design summit took place in San Antonio, Texas, US, which is the county seat for Bexar county.
2진법의
Information that consists solely of ones and zeroes, which is the language of computers.
비트
A bit is a single digit number that is in base of 2 (either a zero or one). Bandwidth usage is measured in bits per second.
초당 비트(BPS)

얼마나 데이터가 빠른지에 대한 보편적인 방법은 여러곳으로 전송된다.

block migration
A method of VM live migration used by KVM to evacuate instances from one host to another with very little downtime during a user-initiated switchover. Does not require shared storage. Supported by Compute.

Block Storage API
